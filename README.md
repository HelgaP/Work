Файлы для мониторинга проектов

Скрипт по отслеживанию проектов позволяет автоматизировано:
1. Определить статус по проектам, своевременно выявить проблему
2. Сформировать отчеты в разрезе каждого проекта для клиентов
3. Сделать рассылку отчетов по почте заинтересованным менеджерам и клиентам
4. Сделать сводную статистику по всем проектам для руководства 
5. Оценить сложность завершенных проектов согласно разработанным критериям

Анализ по проектам за год

В этом скрипте проанализированы проектные поставки за год:
1. Описательный анализ и визуализация по основным бизнесам, продуктам, поставщикам, клиентам 
2. Оценка уровня клиентского сервиса 
3. Root cost analysis основных причин задержек поставок по проектам
